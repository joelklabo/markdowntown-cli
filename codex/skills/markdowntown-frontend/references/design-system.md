# Design system distilled

## Color + tokens
- Do not use raw hex values; use tokens or existing classes.
- Avoid Tailwind neutral palette classes (linted).

## Typography
- Keep type scales consistent with existing components.
- Favor clear hierarchy: strong H1/H2, readable body, short line lengths.

## Spacing + layout
- Use consistent spacing tokens; avoid ad-hoc margins.
- Keep layout grids aligned with existing surfaces.

## Motion
- Use subtle transitions; avoid long sequences.
- Prefer opacity/transform transitions for panel changes.
